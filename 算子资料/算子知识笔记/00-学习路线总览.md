# ClearVision 视觉算子学习路线总览

这套笔记不是按“功能菜单顺序”写的，而是按“一个 0 基础读者真正能学懂的顺序”组织的。建议先把图像怎样变干净、怎样变成前景和边缘学明白，再进入 Blob、轮廓、匹配、测量，最后再碰标定和畸变校正。

## 2. 四个学习模块

### 预处理与分割

- [GaussianBlur](./01-GaussianBlur-技术笔记.md)、[MedianBlur](./02-MedianBlur-技术笔记.md)、[BilateralFilter](./03-BilateralFilter-技术笔记.md): 解决噪声问题，但保留信息的方式不一样。
- [Threshold](./04-Threshold-技术笔记.md)、[AdaptiveThreshold](./05-AdaptiveThreshold-技术笔记.md): 解决前景/背景分离问题。
- [CannyEdge](./06-CannyEdge-技术笔记.md)、[ColorConversion](./08-ColorConversion-技术笔记.md)、[ClaheEnhancement](./09-ClaheEnhancement-技术笔记.md)、[Morphology](./10-Morphology-技术笔记.md): 解决边缘质量、颜色表达、局部对比度和区域清理问题。

### 区域与轮廓

- [BlobDetection](./11-BlobDetection-技术笔记.md): 先把区域找出来，再做面积、圆度、孔洞数等分析。
- [FindContours](./12-FindContours-技术笔记.md): 更偏轮廓层面的提取和几何描述。

### 匹配与测量

- [TemplateMatch](./13-TemplateMatch-技术笔记.md): 灰度模板匹配，适合纹理稳定、姿态变化小的目标。
- [ShapeMatching](./14-ShapeMatching-技术笔记.md): 更强调轮廓和形状，可容忍旋转与尺度变化。
- [CaliperTool](./15-CaliperTool-技术笔记.md)、[CircleMeasurement](./16-CircleMeasurement-技术笔记.md)、[LineMeasurement](./17-LineMeasurement-技术笔记.md): 更偏工程测量，不只是“看见”，而是“量准”。

### 几何校正与标定

- [PerspectiveTransform](./18-PerspectiveTransform-技术笔记.md): 解决平面拍摄角度问题。
- [CameraCalibration](./19-CameraCalibration-技术笔记.md): 估计相机内参和畸变。
- [Undistort](./20-Undistort-技术笔记.md): 使用已有标定结果对图像做校正。

## 3. 推荐的阅读前置

- 如果你刚接触视觉算法，先读 [ColorConversion](./08-ColorConversion-技术笔记.md) 和 [Threshold](./04-Threshold-技术笔记.md)。这两篇能帮你建立“灰度、颜色、前景、背景”这些最基础的概念。
- 如果你已经会做简单二值化，下一步读 [Morphology](./10-Morphology-技术笔记.md) 和 [BlobDetection](./11-BlobDetection-技术笔记.md)。这会让你理解为什么很多工业项目不是“拍了就识别”，而是“先做区域清理，再做区域分析”。
- 如果你关心尺寸和位置精度，优先读 [CannyEdge](./06-CannyEdge-技术笔记.md)、[SubpixelEdgeDetection](./07-SubpixelEdgeDetection-技术笔记.md) 和 [CaliperTool](./15-CaliperTool-技术笔记.md)。
- 如果你关心镜头变形、倾斜拍摄、像素到物理尺寸，优先读 [CameraCalibration](./19-CameraCalibration-技术笔记.md)、[Undistort](./20-Undistort-技术笔记.md) 和 [PerspectiveTransform](./18-PerspectiveTransform-技术笔记.md)。

## 4. 读这些笔记时要一直记住的三件事

1. 教材里的算法原理，不等于仓库里的当前实现。每篇笔记都会先讲 ClearVision 当前实现口径。
2. 一个算子几乎从来不是单独工作的。工业视觉更常见的是“链路”，例如 `Threshold -> Morphology -> BlobDetection`。
3. 很多失败不是算法公式错了，而是上游输入条件不满足。例如光照不稳、ROI 没裁、对比度太低、模板本身选错。

## 5. 建议你怎么做学习实验

- 用同一张图分别试 [GaussianBlur](./01-GaussianBlur-技术笔记.md)、[MedianBlur](./02-MedianBlur-技术笔记.md)、[BilateralFilter](./03-BilateralFilter-技术笔记.md)，感受“去噪”和“保边”的取舍。
- 用亮度不均匀的图同时试 [Threshold](./04-Threshold-技术笔记.md) 和 [AdaptiveThreshold](./05-AdaptiveThreshold-技术笔记.md)，感受全局阈值和局部阈值的差别。
- 用一张带孔洞、带粘连缺陷的二值图试 [Morphology](./10-Morphology-技术笔记.md) 和 [BlobDetection](./11-BlobDetection-技术笔记.md)，感受“前处理决定下游结果”的含义。
- 用同一个目标分别试 [TemplateMatch](./13-TemplateMatch-技术笔记.md)、[ShapeMatching](./14-ShapeMatching-技术笔记.md)、[CaliperTool](./15-CaliperTool-技术笔记.md)，你会很快明白“找相似图块”和“找几何边缘”是两种完全不同的工程思路。
