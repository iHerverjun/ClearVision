# 文档导航

`docs/` 负责计划、任务、门禁说明、验收报告模板与历史闭环材料。

## 目录说明

- [`active/`](./active/)：当前执行中的计划与任务。
- [`reference/`](./reference/)：规范、门禁说明、审计与参考资料。
- [`templates/`](./templates/)：可复用模板（计划、任务、验收报告等）。
- [`closed/`](./closed/)：已闭环专题与阶段报告。
- [`needs-review/`](./needs-review/)：状态待确认文档。
- [`scratch/`](./scratch/)：临时草稿与过程记录。

## Frontmatter约定

- `title`：文档标题
- `doc_type`：文档类型（如 `task-list / plan / report / summary / audit / overview / template / index`）
- `status`：文档状态（`active / needs-review / closed / template`）
- `topic`：专题归属
- `created`：创建日期
- `updated`：最近更新日期
