## 最终验收执行顺序

1. 跑检测专项回归：
```powershell
& "./scripts/run-tests-detection-regression.ps1" -NoBuild -NoRestore -Verbosity minimal
```

2. 跑检测性能门禁（默认自动档位）：
```powershell
& "./scripts/run-tests-detection-performance.ps1" -GateProfile auto -NoBuild -NoRestore -Verbosity minimal
```

3. 验收周强制收紧到 `1.2`（二选一）：
```powershell
# 方式A：显式 acceptance
& "./scripts/run-tests-detection-performance.ps1" -GateProfile acceptance -NoBuild -NoRestore -Verbosity minimal

# 方式B：auto + 时间阈值（到点后自动收紧）
$env:CV_DETECTION_PERF_TIGHTEN_FROM_UTC="2026-05-01T00:00:00Z"
& "./scripts/run-tests-detection-performance.ps1" -GateProfile auto -NoBuild -NoRestore -Verbosity minimal
```

4. 依据测试结果填写四份报告模板。

5. 在 PR 中挂载四报告结论与关键产物路径，按门禁规则判定是否可合入。

## 验收关卡（最终口径）

- 精度：
  - `PointLine/LineLine/Angle/MeasureDistance` 解析场景误差 `<= 0.01px`
  - `Caliper/Width/Gap` MAE `<= 0.15px`，P95 `<= 0.30px`
  - `Circle/Line/Contour` 主指标误差 `<= 0.20px` 或面积相对误差 `<= 1.5%`
- 稳定性：
  - 同图重复 100 次，关键输出标准差 `<= 0.05px`
  - 失败率 `<= 0.5%`（低对比专项 `<= 2%`，单列）
- 性能：
  - 统一统计 `mean/p95/p99`
  - 约束：`p95 <= budget * scale`
  - 常规：`scale=1.5`；验收周：`scale=1.2`
- 回归：
  - 检测专项回归套件全绿
  - 不得降低既有 `PerformanceBudgetAcceptanceTests` 通过率

## 发布判定

- 满足四报告全部 “PASS”：进入发布候选。
- 存在未达项：必须进入“明确遗留清单”，禁止口头“接近”结论。
