## 闪卡 1: CannyEdge完整流程

**Q: Canny边缘检测的5个步骤是什么？**

<details>
<summary>点击查看答案</summary>

**完整流程**:
```
Step 0: 灰度转换（如需要）
    ↓
Step 1: 高斯平滑（去噪）
    ↓
Step 2: 计算梯度（Sobel算子）
    ↓
Step 3: 非极大值抑制（边缘细化）
    ↓
Step 4: 双阈值检测
    ↓
Step 5: 滞后边缘连接
    ↓
输出：边缘图
```

**记忆口诀**: 灰→噪→梯→细→双→连

**各步骤作用**:
1. **高斯平滑**: 抑制噪声
2. **Sobel求梯度**: 找变化最大的地方
3. **非极大值抑制**: 把粗边缘变细
4. **双阈值**: 区分强边缘、弱边缘、噪声
5. **滞后连接**: 强边缘带动的弱边缘保留

</details>

## 闪卡 2: 双阈值策略

**Q: Canny的双阈值策略是如何工作的？**

<details>
<summary>点击查看答案</summary>

**三个区域划分**:
```
梯度强度轴
    ↑
255 |                              ┌──────────┐
    │                              │  强边缘  │ ← 确定保留（>T2）
T2  |──────────────────────────────┤          │
    │          ┌───────────────────┘          │
    │          │      弱边缘区域              │ ← 有条件保留（T1~T2）
T1  |──────────┤                              │
    │          └───────────────────┐          │
    │                              │  噪声    │ ← 直接丢弃（<T1）
  0 |──────────────────────────────┴──────────┘
```

**滞后连接规则**:
- **强边缘**（>T2）: 100%保留
- **弱边缘**（T1~T2）: 如果与强边缘相连则保留，否则丢弃
- **噪声**（<T1）: 直接丢弃

**阈值比例建议**:
```
T1 : T2 ≈ 1 : 2  或  1 : 3

示例: T1 = 50, T2 = 150
```

</details>

## 闪卡 3: CannyEdge参数调参

**Q: CannyEdge的关键参数如何调参？**

<details>
<summary>点击查看答案</summary>

| 参数 | 默认值 | 作用 | 调参建议 |
|------|--------|------|----------|
| **Threshold1** | 50 | 低阈值 | 建议为高阈值的1/2或1/3 |
| **Threshold2** | 150 | 高阈值 | 边缘强度的"及格线" |
| **AutoThreshold** | false | 自动阈值 | 现场波动大时建议开启 |
| **EnableGaussianBlur** | true | 预平滑 | 噪声多必开 |
| **GaussianKernelSize** | 5 | 平滑核大小 | 越大去噪越强，但边缘越模糊 |

**调参速查表**:

| 现象 | 可能原因 | 调节建议 |
|------|----------|----------|
| 边缘太多、杂乱无章 | 阈值太低 / 去噪不够 | 提高T1,T2 / 增大GaussianKernelSize |
| 边缘断裂、不连续 | 阈值太高 / 平滑过度 | 降低T1,T2 / 减小GaussianKernelSize |
| 弱边缘丢失 | 高阈值太高 | 降低T2或开启AutoThreshold |
| 噪声边缘太多 | 低阈值太低 | 提高T1或增强高斯平滑 |

</details>

## 闪卡 4: 亚像素边缘检测

**Q: 什么是亚像素边缘检测？它与普通边缘检测有什么区别？**

<details>
<summary>点击查看答案</summary>

**核心区别**:
- **像素级**: 边缘位置精确到1个像素
- **亚像素级**: 边缘位置精确到0.1像素甚至更高

**示例**:
```
像素级检测结果:                亚像素级检测结果:
    ↓ 检测到的边缘                ↓ 检测到的边缘 120.35
    |                             |
119 |    ○                      119|    ○
    |   /|\                       |   /|\
120 |  / | \  ← 边缘"跳"到整列  120|---●---\  ← 边缘平滑过渡
    | /  |  \                     | /  | ·  \     · 表示亚像素位置
121 |/   ○   \                  121|/   ○    \
```

**为什么需要？**
- 测量10mm零件，图像分辨率1000×1000
- 像素级误差: ±0.1mm
- 亚像素误差: ±0.002mm

**应用场景**: 精密测量（半导体、PCB检测、精密机械）

</details>

## 闪卡 5: 亚像素精度方法

**Q: 亚像素边缘检测的常用方法有哪些？**

<details>
<summary>点击查看答案</summary>

**常用方法**:

| 方法 | 原理 | 适用场景 |
|------|------|----------|
| **抛物线拟合** | 假设灰度分布可用抛物线近似，通过三个点拟合求顶点 | 对称性好的边缘 |
| **高斯拟合** | 假设灰度分布服从高斯函数，取对数后变抛物线 | 更符合真实成像，抗噪更好 |
| **Steger方法** | 使用Hessian矩阵检测线条中心 | 检测细线、激光线、丝状物 |

**抛物线拟合公式**:
```
        y₋₁ - y₁
offset = ───────────
       2(y₋₁ - 2y₀ + y₁)
```

**计算步骤**:
1. 沿梯度方向采样灰度值
2. 用抛物线/高斯拟合
3. 求拟合曲线的极值点位置
4. 得到亚像素偏移量

</details>

## 闪卡 6: 亚像素 vs 像素级选择

**Q: 什么时候必须用亚像素边缘检测？什么时候像素级就够了？**

<details>
<summary>点击查看答案</summary>

**必须用亚像素的场景**:

| 行业 | 应用场景 | 精度要求 |
|------|----------|----------|
| 半导体 | 芯片引脚间距测量 | 引脚间距0.5mm，像素级±0.1mm不可接受 |
| PCB检测 | 焊盘位置、线宽测量 | 线宽公差±0.02mm |
| 精密机械 | 轴承直径、孔径测量 | 直径公差微米级 |
| 3C电子 | 手机屏幕贴合对位 | 对位精度±0.05mm |

**像素级就够的场景**:
- 大目标检测（如整车检测）
- 缺陷粗筛（只需要"有没有"）
- 实时性要求极高（亚像素计算量大）
- 图像分辨率本来就很高

**决策流程**:
```
是否需要精密测量？
   ├── 是 → 用亚像素边缘检测
   └── 否 → 用普通CannyEdge即可
```

</details>

## 闪卡 7: 边缘检测链路

**Q: 写出完整的边缘检测处理链路**

<details>
<summary>点击查看答案</summary>

**像素级边缘检测链路**:
```
原始图像
   ↓
ColorConversion (BGR→Gray)
   ↓
GaussianBlur (去噪，Kernel=5, Sigma=1.0)
   ↓
CannyEdge (Threshold1=50, Threshold2=150)
   ↓
FindContours (提取轮廓)
   ↓
形状分析/测量
```

**亚像素级边缘检测链路**:
```
原始图像
   ↓
GaussianBlur (轻量平滑)
   ↓
CannyEdge (找候选边缘)
   ↓
FindContours (提取轮廓)
   ↓
Sobel求梯度方向
   ↓
沿梯度方向采样
   ↓
抛物线/高斯拟合求亚像素位置
   ↓
SubpixelEdgeDetection输出
   ↓
CaliperTool / 测量工具
```

</details>

## 闪卡 8: 边缘检测常见错误

**Q: 边缘检测失败的常见原因有哪些？**

<details>
<summary>点击查看答案</summary>

**❌ 错误1: 边缘满图都是，杂乱无章**
- **原因**: 阈值太低 / 去噪不够
- **解决**: 提高Threshold1和Threshold2，增大GaussianKernelSize

**❌ 错误2: 边缘断裂、不连续**
- **原因**: 阈值太高 / 平滑过度
- **解决**: 降低T1,T2，减小GaussianKernelSize

**❌ 错误3: 强椒盐噪声干扰边缘检测**
- **原因**: 高斯平滑对椒盐噪声效果差
- **解决**: 先用MedianBlur去椒盐，再用Canny

**❌ 错误4: 亚像素结果飘忽不定**
- **原因**: 上游输入不满足高精度前提（边缘本来就虚）
- **解决**: 改善成像质量，提高对比度

**❌ 错误5: 把Canny检出的边缘当最终测量边**
- **原因**: Canny只是像素级边缘
- **解决**: 精密测量需要SubpixelEdgeDetection或CaliperTool

</details>
