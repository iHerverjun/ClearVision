## 🎯 快速学习路径

### 初学者路径 (按顺序学习)
```

2. 滤波算子闪卡 → 掌握图像平滑技术
3. 阈值分割闪卡 → 掌握图像分割基础

5. 边缘检测闪卡 → 学会提取边缘
6. Blob与轮廓闪卡 → 学会区域分析
7. 模板与形状匹配闪卡 → 学会目标识别
```

### 关键参数速查

| 算子 | 关键参数 | 默认值 | 调参方向 |
|------|----------|--------|----------|
| GaussianBlur | KernelSize, Sigma | 5, 1.0 | 噪声大则增大 |
| MedianBlur | KernelSize | 5 | 椒盐噪声用中值 |
| Threshold | Thresh, Maxval | 127, 255 | 根据直方图双峰 |
| CannyEdge | Threshold1, Threshold2 | 50, 150 | 比例1:2或1:3 |
| Morphology | KernelSize | 3 | 根据特征大小 |
| TemplateMatch | Threshold | 0.8 | CCoeffNormed>0.8 |
| ShapeMatching | AngleStep, ScaleStep | 2°, 0.1 | 精度vs速度权衡 |

## 💡 使用建议

### 学习方法
1. **第一轮**: 通读所有闪卡，建立整体概念
2. **第二轮**: 遮住答案，自我测试
3. **第三轮**: 针对薄弱点重点复习
4. **实战中**: 遇到问题时快速查阅对应闪卡

### 常见误区提醒
- ❌ 把透视畸变当镜头畸变
- ❌ HSV的H范围用错（记住是0-179）
- ❌ 直接用Canny边缘做精密测量
- ❌ 椒盐噪声用高斯滤波处理
- ❌ 模板匹配不支持旋转/缩放
