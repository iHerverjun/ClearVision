## 闪卡 1: CaliperTool核心功能

**Q: CaliperTool算子是什么？它的核心功能是什么？**

<details>
<summary>点击查看答案</summary>

**核心功能**:
沿一条线段扫描像素灰度变化，找出边缘位置，用于测量两点间距离。

**类比理解**:
想象用游标卡尺卡住目标物体的两边 → Caliper就是电子版的"卡尺"

**工作原理**:
```
搜索区域（ROI）
┌──────────────────────────────────┐
│  ○ ← 起点                        │
│    \                             │
│     \                            │
│      \ ← 扫描线（与边缘垂直）    │
│       \                          │
│        ○ ← 终点                  │
│                                  │
│    ██████████                    │
│    ██████████ ← 目标边缘         │
│    ██████████                    │
└──────────────────────────────────┘
```

**典型应用**:
- 测量零件宽度、高度
- 测量两边缘间距离
- 定位边缘位置

</details>

## 闪卡 2: Caliper参数详解

**Q: CaliperTool的关键参数有哪些？如何调参？**

<details>
<summary>点击查看答案</summary>

**关键参数**:

| 参数 | 作用 | 设置建议 |
|------|------|----------|
| **EdgeThreshold** | 灰度差阈值 | 设为边缘两侧灰度差的一半以上 |
| **EdgePolarity** | 边缘极性 | Rising(黑→白) / Falling(白→黑) / Any |
| **SearchDirection** | 搜索方向 | 从起点到终点扫描 |
| **Smoothing** | 平滑处理 | 噪声多时开启 |
| **EdgePair** | 寻找边缘对 | 测量宽度时开启 |

**极性选择**:
```
Rising (黑→白)   : 背景黑，物体白
Falling (白→黑)  : 背景白，物体黑
Any              : 不关心，两种情况都找
```

**调参流程**:
```
1. 观察灰度剖面图，确定边缘阈值
2. 设置EdgeThreshold（灰度差的一半以上）
3. 设置正确极性
4. 开启Smoothing（如有噪声）
5. 测试调整
```

</details>

## 闪卡 3: 圆测量原理

**Q: CircleMeasurement是如何测量圆的？**

<details>
<summary>点击查看答案</summary>

**测量原理 - 三点定圆**:
```
在圆周上找多个边缘点 → 拟合圆 → 得到圆心(x,y)和半径r

数学原理:
圆的方程: (x-a)² + (y-b)² = r²
        ↓
用最小二乘法拟合所有边缘点
        ↓
输出: 圆心(a,b), 半径r, 拟合误差
```

**关键点**:
- 至少需要3个点才能确定一个圆
- 边缘点越多，拟合越稳定
- 拟合误差可反映圆的质量

**鲁棒圆拟合**:
```
标准圆拟合: 所有点参与拟合
鲁棒圆拟合: 用RANSAC等算法，排除离群点

优势: 抗干扰能力强，即使部分点错误也能得到好结果
```

</details>

## 闪卡 4: 透视变换 vs 畸变校正

**Q: PerspectiveTransform和Undistort有什么区别？**

<details>
<summary>点击查看答案</summary>

**核心区别**:

| 特性 | PerspectiveTransform | Undistort |
|------|---------------------|-----------|
| **校正对象** | 透视畸变（投影畸变） | 镜头畸变（径向/切向） |
| **畸变原因** | 相机视角倾斜 | 镜头光学缺陷 |
| **数学模型** | 透视投影（3×3单应矩阵） | 径向畸变模型（k1,k2,k3） |
| **是否需要标定** | 不需要（已知4点对应） | 需要相机标定（获取参数） |
| **典型表现** | 近大远小、矩形变梯形 | 直线变弯曲、桶形/枕形 |

**常见误区**:
```
❌ 把透视畸变当镜头畸变
❌ 认为Undistort也能解决透视问题
❌ 用透视变换校正镜头畸变
```

**正确区分**:
- **Undistort** → 让弯曲的直线变直（镜头问题）
- **PerspectiveTransform** → 让梯形变矩形（视角问题）

</details>

## 闪卡 5: 透视变换

**Q: 如何使用PerspectiveTransform进行透视校正？**

<details>
<summary>点击查看答案</summary>

**四点对应原理**:
```
源图像（梯形）                目标图像（矩形）
    A ─────── B                  A'───────B'
    /           \                 │          │
   /             \       →        │          │
  D ──────────── C                D'───────C'

只需要4个点的对应关系，就能计算出变换矩阵
```

**两种应用场景**:

**场景1: 已知目标坐标**
```
4个角点在原图中的像素坐标 → 变换到标准坐标系
（如标准A4纸的尺寸）
```

**场景2: 仅校正矩形**
```
4个角点 → 变换为轴对齐矩形
（最常见：把倾斜的物体摆正）
```

**ClearVision参数**:
```
PerspectiveTransform{
    srcPoints: [ (x0,y0), (x1,y1), (x2,y2), (x3,y3) ],
    dstPoints: [ (x0',y0'), (x1',y1'), (x2',y2'), (x3',y3') ]
}
```

**要点**:
- 4个点不能共线
- 点序要保持一致（顺时针或逆时针）

</details>

## 闪卡 6: 相机标定

**Q: 相机标定的目的是什么？标定后得到哪些参数？**

<details>
<summary>点击查看答案</summary>

**标定目的**:
1. 得到相机的内参 → 用于畸变校正（Undistort）
2. 得到相机的外参 → 用于世界坐标→像素坐标转换
3. 建立像素坐标与实际物理尺寸的映射

**标定结果参数**:

| 参数 | 含义 | 用途 |
|------|------|------|
| **fx, fy** | x/y方向焦距 | 3D重建、位姿估计 |
| **cx, cy** | 光心坐标 | 相机内参矩阵 |
| **k1, k2, k3** | 径向畸变系数 | 畸变校正 |
| **p1, p2** | 切向畸变系数 | 畸变校正 |
| **R, t** | 旋转/平移矩阵 | 外参，多相机标定用 |

**ClearVision支持两种标定**:
- **单相机标定**: 输出内参
- **双目标定**: 输出内参+外参+基线

</details>

## 闪卡 7: 畸变校正

**Q: Undistort如何使用？如何选择畸变模型？**

<details>
<summary>点击查看答案</summary>

**使用步骤**:
```
Step 1: 相机标定
  拍摄棋盘格图像 → 计算畸变参数 (k1,k2,k3,p1,p2)

Step 2: 应用校正
  输入畸变图像 + 畸变参数 → Undistort → 校正图像
```

**畸变模型选择**:

| 模型 | 包含参数 | 适用情况 |
|------|----------|----------|
| **简单模型** | k1, k2 | 广角、鱼眼镜头 |
| **标准模型** | k1, k2, p1, p2 | 普通镜头 |
| **完整模型** | k1, k2, k3, p1, p2 | 广角、畸变严重 |

**常见错误**:
```
❌ 分不清k1k2是径向还是切向
  → 记住: k开头=径向，p开头=切向

❌ 标定只跑1-2张图
  → 至少15-20张，覆盖各角度、位置

❌ 用畸变系数做透视校正
  → 两种校正独立进行
```

</details>

## 闪卡 8: 测量完整链路

**Q: 设计一个完整的精密测量流程**

<details>
<summary>点击查看答案</summary>

**完整测量链路**:
```
原始图像
   ↓
Undistort (畸变校正 - 如果需要高精度)
   ↓
PerspectiveTransform (透视校正 - 如果视角倾斜)
   ↓
GaussianBlur (轻量去噪)
   ↓
边缘检测/模板匹配 (定位测量区域)
   ↓
CaliperTool (沿特定方向扫描边缘)
   ↓
   ├── 线测量 → LineMeasurement (拟合直线)
   └── 圆测量 → CircleMeasurement (拟合圆)
   ↓
输出测量结果 (位置、尺寸、误差)
```

**关键要点**:
1. **校正优先**: 先处理畸变和透视，再进行测量
2. **定位要准**: 用匹配算法找到测量ROI
3. **边缘要稳**: 确保边缘检测参数合适
4. **拟合要准**: 足够多的边缘点参与拟合

**精度保障**:
- 使用亚像素边缘检测
- 多角度采样求平均
- 多次测量取均值

</details>
