## 闪卡 1: 什么是像素？

**Q: 数字图像中的"像素"是什么？**

<details>
<summary>点击查看答案</summary>

像素（Pixel）是数字图像的最小单位，相当于一张方格纸上的每个格子。

- 灰度图像：每个像素有1个数值（0-255），0=纯黑，255=纯白
- 彩色图像：每个像素有3个数值（R,G,B各0-255）

**记忆口诀**: 像素就像拼图块，组成整个画面。

</details>

## 闪卡 2: 图像分辨率

**Q: 一张800×600的灰度图有多少个像素？占用多少字节？**

<details>
<summary>点击查看答案</summary>

- **像素数量**: 800 × 600 = 480,000 像素（48万像素）
- **字节数**: 480,000 字节 ≈ 468.75 KB

计算公式：`宽度 × 高度 × 每像素字节数`
- 灰度图：1字节/像素
- 彩色图：3字节/像素

</details>

## 闪卡 3: 卷积是什么？

**Q: 图像处理中的"卷积"是什么？**

<details>
<summary>点击查看答案</summary>

卷积 = 一个"小窗口"在图像上滑动，每滑一步就对窗口内的像素进行加权计算。

**比喻**: 像拿着一个放大镜在图片上慢慢移动，每次只看局部区域并计算。

**核心要素**:
- **卷积核/滤波器**: 小窗口内的权重矩阵
- **滑动步长**: 每次移动多少像素
- **边界处理**: 边缘像素如何补齐

</details>

## 闪卡 4: 前景 vs 背景

**Q: 图像分割中的"前景"和"背景"分别指什么？**

<details>
<summary>点击查看答案</summary>

| 概念 | 说明 | 示例 |
|------|------|------|
| **前景** | 我们感兴趣的目标物体 | 白纸上的黑字、传送带上的零件 |
| **背景** | 不感兴趣的衬托区域 | 白纸本身、传送带表面 |

**关键前提**: 阈值分割的前提是前景和背景在亮度上有明显差异。

</details>

## 闪卡 5: 噪声类型对比

**Q: 高斯噪声和椒盐噪声有什么区别？分别用什么滤波器处理？**

<details>
<summary>点击查看答案</summary>

| 特性 | 高斯噪声 | 椒盐噪声 |
|------|----------|----------|
| **视觉表现** | 像蒙了一层细纱，颗粒细腻 | 像撒了黑白胡椒，脏点明显 |
| **数值特征** | 连续变化，符合正态分布 | 只有0或255（极端值） |
| **产生原因** | 传感器热噪声、电路噪声 | 传输错误、传感器坏点 |
| **最佳处理** | **高斯模糊** ✅ | **中值滤波** ✅ |

**禁忌**: 椒盐噪声 + 高斯模糊 = 更大的灰斑灾难！

</details>

## 闪卡 6: 梯度是什么？

**Q: 图像处理中的"梯度"是什么意思？**

<details>
<summary>点击查看答案</summary>

梯度 = 灰度值变化的快慢和方向。

**生活比喻**: 爬山时的坡度
- **陡坡**: 每走一步海拔变化大 → 梯度大 → 可能是边缘
- **缓坡**: 走很远海拔才变 → 梯度小 → 平缓区域
- **平地**: 海拔不变 → 梯度为零 → 平坦区域

**在图像中**:
- 从黑突然跳到白 → 梯度大 → 边缘！
- 灰度缓慢过渡 → 梯度小 → 阴影或纹理
- 同一块颜色内部 → 梯度为零 → 平坦区域

</details>

## 闪卡 7: 连通域 4连通 vs 8连通

**Q: 4连通和8连通有什么区别？**

<details>
<summary>点击查看答案</summary>

**4连通**（四连通）:
```
  ↑
← ● →    水只能向上下左右4个方向流动
  ↓
```

**8连通**（八连通）:
```
↖ ↑ ↗
← ● →    水可以向8个方向流动（包括对角线）
↙ ↓ ↘
```

**实际影响**:
```
黑 白 白 黑
n白 黑 黑 白
黑 白 白 黑

4连通判断: 两条独立的线（对角不相连）
8连通判断: 连在一起形成一个X形
```

ClearVision默认使用**8连通**。

</details>

## 闪卡 8: 颜色空间对比

**Q: RGB、HSV、Lab三种颜色空间各适合什么场景？**

<details>
<summary>点击查看答案</summary>

| 颜色空间 | 适用场景 | 特点 |
|----------|----------|------|
| **RGB/BGR** | 显示器显示、图像存储 | 硬件控制，但不适合颜色筛选 |
| **HSV** | 颜色筛选、颜色分类 | H通道分离色相，光照变化时稳定 |
| **Lab** | 亮度调整、色差计算 | L通道完全独立，修改不影响颜色 |

**快速记忆**:
- 要**提取红色物体** → 用 **HSV**
- 要**只调亮度不改颜色** → 用 **Lab**
- 要**显示图像** → 用 **RGB/BGR**

</details>

## 闪卡 10: 算子链路记忆

**Q: 写出完整的预处理链路（从图像输入到Blob检测）**

<details>
<summary>点击查看答案</summary>

**标准链路**:
```
原始图像
   ↓
[ColorConversion]  彩色→灰度
   ↓
[GaussianBlur]     去噪（Kernel=5, Sigma=1.0）
   ↓
[AdaptiveThreshold] 二值化（BlockSize=11, C=2）
   ↓
[Morphology]       开运算去噪/闭运算填孔
   ↓
[BlobDetection]    检测连通域并计算特征
```

**记忆口诀**: 转灰→去噪→分割→清理→检测

</details>
