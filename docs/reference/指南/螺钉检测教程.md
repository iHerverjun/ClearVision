## 场景介绍

### 🔩 实际生产场景

假设你是一家五金厂的质检员，每天要检查数千颗螺钉是否合格。传统人工检测效率低、容易疲劳出错。

**检测目标**：
- ✅ 螺钉头部是否完整（无缺损）
- ✅ 螺钉长度是否符合标准
- ✅ 螺纹是否清晰

**预期效果**：
- 每颗螺钉检测时间 < 100ms
- 自动区分 OK（合格）/ NG（不合格）
- 保存检测结果便于追溯

## 步骤二：搭建检测流程

### 🧩 流程设计思路

一个完整的视觉检测流程通常包含四个阶段：

```
图像采集 → 图像预处理 → 检测分析 → 结果输出
    │           │            │          │
    ▼           ▼            ▼          ▼
 读入图片    去噪/增强    找特征/测量  判定OK/NG
```

### 2.1 第一步：图像采集

**作用**：告诉系统从哪里获取图像

1. 在左侧算子库中找到 **"图像处理"** 分类
2. 找到 `ImageAcquisition`（图像采集）算子
3. **按住鼠标左键**，将它拖拽到中间画布上

![拖拽算子](docs/images/tutorial/drag_operator.gif)

4. **双击**画布上的 ImageAcquisition 算子，打开属性面板

#### 配置参数

```yaml
源类型: 文件        # 选择"文件"（因为我们是离线测试）
文件路径:           # 点击"浏览"按钮，选择你的螺钉图片
```

![配置图像采集](docs/images/tutorial/config_acquisition.png)

> 💡 **知识卡片**：
> - **源类型=文件**：从本地图片读取，适合离线调试
> - **源类型=相机**：连接工业相机，适合在线检测

### 2.2 第二步：图像预处理（可选但推荐）

**作用**：让图像更清晰，便于后续检测

#### 添加高斯模糊（去噪）

1. 从算子库拖拽 `GaussianBlur`（高斯模糊）到画布
2. **连接两个算子**：
   - 从 `ImageAcquisition` 的**右侧输出口**（绿色）拖拽
   - 连接到 `GaussianBlur` 的**左侧输入口**（蓝色）

![连接算子](docs/images/tutorial/connect_operators.gif)

3. 双击 `GaussianBlur` 配置参数：

```yaml
核大小: 5      # 奇数，越大模糊效果越强
Sigma: 1.5     # 标准差，通常 1.0~3.0
```

> 💡 **知识卡片**：高斯模糊可以去除图像中的细小噪点，让边缘更清晰。

### 2.3 第三步：检测分析（核心步骤）

#### 方案 A：使用 Blob 分析（推荐初学者）

**适用场景**：螺钉头部完整、形状规则

1. 拖拽 `BlobDetection`（Blob 分析）算子到画布
2. 从 `GaussianBlur` 的输出连接到 `BlobDetection` 的输入
3. 双击配置：

```yaml
最小面积: 5000          # 根据实际图像调整，单位是像素
最大面积: 50000         # 排除过大的干扰物
最小圆度: 0.7           # 0~1，越接近1表示越圆
阈值方法: OTSU          # 自动计算最佳阈值
```

![Blob 分析配置](docs/images/tutorial/config_blob.png)

#### 方案 B：使用深度学习（需要模型）

**适用场景**：缺陷类型复杂、需要高准确率

1. 拖拽 `DeepLearning` 算子到画布
2. 连接输入
3. 配置：

```yaml
模型路径: models/screw_defect.onnx    # YOLO 格式模型
置信度阈值: 0.5                        # 低于此值的结果会被忽略
类别标签:                             # 自定义类别名称
  - defect_head                       # 头部缺陷
  - defect_thread                     # 螺纹缺陷
```

> ⚠️ **注意**：使用深度学习需要提前准备训练好的模型。初学者建议先使用 Blob 分析入门。

### 2.4 第四步：结果输出

**作用**：汇总检测结果，判定 OK/NG

1. 拖拽 `ResultOutput`（结果输出）算子到画布
2. 从 `BlobDetection` 连接输入
3. 配置判定逻辑：

```yaml
判定条件: Blob数量 > 0 AND Blob圆度 >= 0.8
OK 输出: 合格
NG 输出: 缺陷
```

### 2.5 完整流程示意

你的画布现在应该像这样：

```
┌──────────────────────────────────────────┐
│                                          │
│  ┌─────────────┐    ┌─────────────┐     │
│  │ 图像采集    │───▶│  高斯模糊   │     │
│  │ ImageAcq    │    │ GaussianBlur│     │
│  └─────────────┘    └──────┬──────┘     │
│                            │            │
│                            ▼            │
│                     ┌─────────────┐     │
│                     │ Blob 分析   │     │
│                     │ BlobDetect  │     │
│                     └──────┬──────┘     │
│                            │            │
│                            ▼            │
│                     ┌─────────────┐     │
│                     │ 结果输出    │     │
│                     │ ResultOut   │     │
│                     └─────────────┘     │
│                                          │
└──────────────────────────────────────────┘
```

### 2.6 保存工程

按 `Ctrl+S` 保存当前工程。

## 步骤三：运行检测

### 3.1 切换到检测视图

点击顶部导航栏的 **"检测"** 选项卡。

### 3.2 运行单次检测

1. 点击工具栏上的 **▶ 运行** 按钮（或按 `F5`）
2. 观察右侧的结果面板

#### 预期结果

**OK 样本**：
```
✅ 检测状态: OK
📊 检测数量: 1
🎯 置信度: 98.5%
📐 Blob面积: 15234 像素
⏱️ 耗时: 45ms
```

**NG 样本**（头部缺损）：
```
❌ 检测状态: NG
📊 检测数量: 0
⚠️ 失败原因: 未找到符合条件的Blob
⏱️ 耗时: 38ms
```

![检测结果展示](docs/images/tutorial/result_panel.png)

- **绿色方框**：检测到的目标
- **红色标记**：NG 原因标注
- **黄色文字**：测量数值

1. 回到"流程"选项卡

3. 重复运行测试

## 步骤四：优化与调整

### 4.1 常见问题排查

#### 问题 1：找不到螺钉（Blob 数量为 0）

**可能原因**：
- ❌ 阈值设置过高，螺钉被过滤掉
- ❌ 面积范围设置不当

**解决方案**：
```yaml
# 尝试降低阈值要求
最小面积: 改为更小的值（如 1000）
最大面积: 改为更大的值（如 100000）

# 或使用自适应阈值
阈值方法: 改为 OTSU（自动）
```

#### 问题 2：检测到多个目标

**可能原因**：
- ❌ 背景干扰
- ❌ 图像中有其他物体

**解决方案**：
```yaml
# 增加面积限制
最小面积: 适当增大，排除小噪点

# 或添加 ROI（感兴趣区域）
使用 ROI: 启用
ROI 区域: 框选出螺钉大概位置
```

#### 问题 3：结果不稳定（同一张图有时 OK 有时 NG）

**可能原因**：
- ❌ 光照变化导致图像亮度不一致
- ❌ 阈值设置过于临界

**解决方案**：
```yaml
# 添加预处理步骤
1. HistogramEqualize（直方图均衡化）- 统一亮度
2. 或调整阈值范围，留有余量
```

### 4.2 参数调优技巧

**面积参数调优**：
1. 先用一张 OK 样本运行检测
2. 记录检测到的 Blob 面积（如 15234）
3. 设置：
   - 最小面积 = 15234 × 0.7 ≈ 10663
   - 最大面积 = 15234 × 1.3 ≈ 19804

**圆度参数调优**：
- 圆形螺钉头：0.8 ~ 1.0
- 六角螺钉头：0.6 ~ 0.9
- 异形头部：0.5 ~ 1.0

### 4.3 添加更多检测项

当基础检测稳定后，你可以扩展：

```
图像采集 → 高斯模糊 → Blob分析 → 圆测量 ──┐
                                          ├→ 结果输出
           └→ 直方图均衡 → Canny边缘 ────┘
```

**新增算子**：
- `CircleMeasurement`：精确测量螺钉头直径
- `LineMeasurement`：测量螺钉总长度
- `ColorDetection`：检测表面是否有锈蚀（颜色异常）

## 💡 常见问题 FAQ

**Q: 我没有工业相机，能用吗？**  
A: 完全可以！教程使用的是"文件"模式，你只需要准备图片即可。

**Q: 检测速度慢怎么办？**  
A: 尝试：1) 降低图像分辨率 2) 减少算子数量 3) 关闭不必要的可视化

**Q: 如何学习更多算子？**  
A: 点击算子库中的算子名称，会显示简要说明。详细文档参见 [算子手册](../../../算子资料/算子手册.md)。

**Q: 可以检测其他产品吗？**  
A: 可以！ClearVision 是通用平台，只需更换图像和调整参数即可检测各类工业产品。
