ClearVision 是一个基于 .NET 8 的工业视觉检测平台。项目采用 `WinForms + WebView2` 桌面宿主与前后端分层架构，支持通过拖拽式流程编辑器编排图像处理、检测、测量、识别与工业通信流程。

## 核心能力
- 拖拽式流程编排：通过节点和连线快速搭建检测流程。
- 丰富算子库：覆盖图像预处理、几何测量、OCR/条码识别、深度学习推理、流程控制等。
- AI 辅助：支持 AI 生成检测工作流与自动布局。
- 工业通信集成：支持 PLC/设备通信（如 Modbus、TCP、串口等）。
- 结果管理：支持检测历史、统计分析、报表导出。
- 本地化运行：桌面端内置本地 Web API 与 SQLite 存储。

## 技术栈
- 后端：`.NET 8`、`C# 12`
- 桌面宿主：`WinForms`、`WebView2`
- 图像处理：`OpenCvSharp`
- 深度学习推理：`ONNX Runtime`
- OCR/条码：`PaddleOCRSharp`、`ZXing.Net`
- 数据存储：`Entity Framework Core` + `SQLite`
- 测试：`xUnit`、`FluentAssertions`、`Playwright`

## 项目结构
```text
Acme.Product/
  src/
```

## 快速开始
### 1. 环境要求
- Windows 10/11 (x64)
- .NET 8 SDK
- WebView2 Runtime（Windows 通常已预装）

### 2. 构建与运行
```bash
cd Acme.Product
dotnet restore
dotnet build Acme.Product.sln
dotnet run --project src/Acme.Product.Desktop/Acme.Product.Desktop.csproj
```

### 3. 登录
首次启动会自动创建默认管理员账号：
- 用户名：`admin`
- 密码：`admin123`

## 测试
### .NET 测试
```bash
cd Acme.Product
dotnet test tests/Acme.Product.Tests/Acme.Product.Tests.csproj
```

### 前端 E2E（Playwright）
```bash
cd Acme.Product/tests/Acme.Product.UI.Tests
npm install
npx playwright install
npx playwright test
```

## 发布
使用内置脚本发布 Win-x64 单文件版本：
```bash
cd Acme.Product/scripts
publish.bat
```
发布产物默认输出到仓库上级目录的 `publish/`。

## 关键配置
- AI 生成配置（初始来源）：`Acme.Product/src/Acme.Product.Desktop/appsettings.json`
- AI 运行时配置（自动生成）：`ai_config.json`（位于应用运行目录）
- 数据库文件：`vision.db`
- 流程 JSON 存储：`App_Data/ProjectFlows/`

## 安全建议
- 首次登录后请立即修改默认管理员密码。
- 请勿在仓库中提交真实 API Key，建议改用本地配置或环境变量注入。
