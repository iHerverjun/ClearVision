# Closed 索引

> 需要安装 Obsidian Dataview 插件。

```dataview
TABLE WITHOUT ID
  file.link AS 文档,
  topic AS 主题,
  doc_type AS 类型,
  updated AS 更新
FROM "docs/closed"
WHERE status = "closed" AND doc_type != "index"
SORT topic ASC, updated DESC, file.name ASC
```
